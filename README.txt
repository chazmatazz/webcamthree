webcamthree
https://github.com/chazmatazz/webcamthree
JavaScript/Flash library for webcam output. Interfaces with Flash AS3. 
Includes black and white output, useful for tracking fiducials.

OVERVIEW

Instructions: Put index.html on a web server and open it.

A swf is included. To build from scratch, you need make and the mxmlc
compiler (Flex API).

Source for flash is in as3_src.

Source for the javascript interface is in js.

An example is in index.html.

The objective of this library is fast streaming of webcam data to JavaScript.
The JavaScript sets a timer that polls flash.
